Lighthouse - Crowd-funding on the block chain

How does it work?

Download

Lighthouse is a brand new approach to all-or-nothing fund raising. Using advanced features of the Bitcoin protocol it is now possible to create Kickstarter-style fund raising initiatives in an entirely decentralised way. No middlemen unless you want them, and no fees beyond the usual Bitcoin transaction fees (just a fraction of a dollar cent).
